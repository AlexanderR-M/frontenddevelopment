Intorduccion

En esta pagina voy hablar sobre mis proyectos de programación y las distintas aplicaciones que estamos utilizando

Visual Studio Code

Es un editor de código fuente gratuito y ligero desarrollado por Microsoft para Windows, macOS y Linux. Es popular entre los desarrolladores por ser personalizable, extensible y por incluir soporte para múltiples lenguajes de programación como JavaScript, Python y C++, entre otros.

HTML

Es el lenguaje estándar para estructurar el contenido de las páginas web. Utiliza etiquetas para definir y organizar elementos como texto, imágenes y enlaces, creando la estructura fundamental de cualquier sitio web que se visualiza en un navegador

CSS

Es un lenguaje que se utiliza para dar estilo y presentación a los documentos web, como los creados con HTML. En lugar de dar formato al contenido directamente, CSS permite separar el contenido de su diseño, controlando así aspectos como el color, el tamaño y la fuente del texto, la disposición de los elementos en la página, los fondos y los efectos visuales.

Java Fundamentals

Conjunto de conocimientos básicos sobre el lenguaje de programación Java, que introduce los conceptos de la programación orientada a objetos (POO) (clases, objetos, herencia, etc.), la sintaxis del lenguaje, y el proceso de creación de programas simples y aplicacione

Ejemplos

Estamos viendo muchas cosas en html, por ejemplo semantic tags y tag examples

Eclipse

Es una plataforma y entorno de desarrollo integrado (IDE) de código abierto que permite crear aplicaciones de software, especialmente en Java, pero también en otros lenguajes como C++, Python o PHP, a través de un sistema de plugins.

SQL

Es un lenguaje de programación estándar utilizado para gestionar y manipular datos en bases de datos relacionales. Permite realizar operaciones como crear, leer, actualizar y eliminar datos (CRUD), además de gestionar la estructura de las bases de datos, lo que lo convierte en una herramienta esencial para el análisis y la administración de datos
